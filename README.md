
⚙️ Sistema de Controle e Inspeção de Qualidade (ERP-CLI)
Este é um software desenvolvido em JavaScript que simula o backend de um sistema de auditoria e controle de processos industriais. O objetivo deste projeto é demonstrar habilidades práticas de desenvolvimento de software, arquitetura de dados e regras de negócio.

🚀 Funcionalidades do Software
Entrada e Validação de Dados: O sistema valida os tipos de dados inseridos, impedindo que inputs corrompidos ou errados quebrem a aplicação.
Processamento de Regras de Negócio: Realiza cálculos automáticos de desvio dimensional ou técnico e define o status do item (Aprovado/Reprovado).
Persistência em Memória: Simula a estrutura de um banco de dados utilizando arrays e objetos estruturados.
Módulo de Relatórios: Um loop avançado varre os dados armazenados e gera uma saída formatada para o usuário no terminal.
🛠️ Conceitos de Programação Aplicados
Estruturas de Dados: Manipulação de Objetos (JSON) e Arrays.
Métodos de Array: Uso do .push() para inserção e .forEach() para iteração e renderização de dados.
Manipulação de Datas: Uso do objeto nativo New Date() do JavaScript para carimbar o momento do registro.
📦 Como rodar a aplicação
Caso tenha o Node.js instalado, basta clonar o repositório e executar no terminal:

node app.js
